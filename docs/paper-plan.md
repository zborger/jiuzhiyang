## 三、实验设计

### 3.1 实验组 vs 对照组

| 组别 | 人数 | 条件 |
|------|------|------|
| 实验组 | 张俱扬所带班级（约40人） | 使用 AI 化学助教 + 常规暑假作业 |
| 对照组 | 同年级另一班级（约40人） | 仅常规暑假作业，不使用平台 |

### 3.2 变量控制
- 两组使用相同的暑假作业
- 两组同一教材版本
- 开学后使用同一份摸底考试卷

## 四、数据收集计划

### 4.1 平台使用数据（自动收集）

| 数据项 | 说明 | 收集方式 |
|--------|------|----------|
| 总提问次数 | 暑假期间所有问题总数 | 前端日志 |
| 人均提问次数 | 每个学生平均问了多少问题 | 前端日志 |
| 热门知识点 TOP10 | 最常被问到的化学知识点 | 关键词统计 |
| 使用时段分布 | 几点使用最多 | 时间戳分析 |
| 满意度评分 | 👍/(👍+👎) 比率 | 反馈按钮 |
| 日活跃用户数 | 每天有多少人使用 | 日志去重 |

### 4.2 问卷调查（暑假结束后发）

**实验组问卷（使用后）：**
1. 你暑假使用AI助教的频率？（每天/每周几次/偶尔/没用过）
2. AI助教对你暑假化学复习帮助大吗？（5分制）
3. 你最常用AI助教解决什么问题？（多选：概念理解/做题讲解/公式记忆/实验原理/其他）
4. 与传统查课本/问老师相比，AI助教的优势是？（开放题）
5. AI助教的回答你能看懂吗？（总能/大部分/一半/很少）
6. 你会推荐同学使用吗？（会/不会/看情况）

**对照组问卷：**
1. 暑假遇到化学问题你怎么解决？（翻课本/问同学/网上搜/不管了）
2. 如果有AI能24小时回答化学问题，你想用吗？

### 4.3 成绩对比

| 测试 | 时间 | 用途 |
|------|------|------|
| 期末考试成绩 | 暑假前 | 基线数据（两组起点是否一致） |
| 开学摸底考 | 9月初 | 实验后数据（暑假学习效果） |

对比指标：
- 平均分差异
- 进步幅度差异
- 各知识点得分率差异（尤其看AI被问最多的知识点）

## 六、学生投放方案

### 6.1 发布方式
- 在班级群发送网站链接（浏览器打开）
- 附使用说明：只需打开网站 → 点"AI助教" → 输入问题
- 强调：免费、24小时、不记名

### 6.2 激励机制
- 暑假开学后展示"热门问题排行榜"
- 表扬使用积极的同学
- 不强制使用，保持自愿性（论文伦理要求）

### 6.3 注意事项
- 不收集学生个人信息（隐私保护）
- 日志只记录问题内容和时间，不记录身份
- 使用前告知学生这是教学研究项目（知情同意）

## 九、风险与应对

| 风险 | 应对措施 |
|------|----------|
| 学生不用 | 在群里定期提醒，展示有趣问答示例 |
| DeepSeek 余额用完 | 10块钱很够用，必要时再充10块 |
| AI 回答不准确 | 系统prompt约束 + 学生满意度反馈 + 后续优化 |
| 网站打不开 | 浏览器直接打开OK，微信问题让学生复制链接到浏览器 |
| 对照组也偷偷用 | 可接受，只要有数据差异即可论证 |
| API Key 泄露 | MVP阶段可接受，设消费上限（DeepSeek支持） |
